# 猛兽派对｜猛兽选择器

## 内容简介

​	本文主要设计了一款根据游戏《猛兽派对》各个角色进行分类的python程序，提供玩家完成每周任务、大逃杀等多种游戏需要，并设计了较为优美的操作界面，便于玩家选择、查看。*（版本 2024.11.06）
